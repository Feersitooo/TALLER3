# TALLER3
el taller 3 nos mete en un mundo de magos y hechizos, donde nuestra mision principal es lograr hacer menus para un administrador y menus para ver las estadisticas de los magos y los hechizos, consta de 2 menus uno de administrador que te permite cambiar cosas de los documentos de texto y otro de analisis que te permite ver las estadisticas de los magos y los hechizos.
